### Creating a store

Olik supports arbitrarily deep state-trees, however just like with Redux, [**normalizing**](https://redux.js.org/recipes/structuring-reducers/normalizing-state-shape) your state is still advised.

<Tabs groupId="create">
<TabItem value="no-types" label="With inferred types">

```ts
import { createStore } from 'olik'

export const store = createStore({
  name: document.title,            // can be any user-defined string
  state: { hello: 'world' }        // can be any plain Javascript object
})
```

</TabItem>
<TabItem value="with-types" label="With declared types">

```ts
import { createStore } from 'olik'

export interface State {
  hello: string;
}

export const store = createStore<State>({
  name: document.title,            // can be any user-defined string
  state: { hello: 'world' }        // can be any plain Javascript object
})
```

</TabItem>
</Tabs>

### Integrating with Redux Devtools

You can integrate with the [**Redux Devtools**](https://github.com/reduxjs/redux-devtools) by importing the related module before you create your store.

```ts
import { importOlikReduxDevtoolsModule } from 'olik'

importOlikReduxDevtoolsModule()
```
